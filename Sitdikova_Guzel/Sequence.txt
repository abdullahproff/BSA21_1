@startuml

skin rose

autonumber

actor User
participant MobileApp 
participant Backend 
Database DB

activate User

User -> MobileApp: Нажимает отправить корзину с товарами  другому пользователю
activate MobileApp
MobileApp -> Backend: http-запрос на получение ссылки на корзину
activate Backend

Backend -> DB: извлекает данные корзины из базы данных
activate DB
DB --> Backend:генерирует ссылку для доступа к корзине
deactivate DB

Backend --> MobileApp: http ответ с адресом ссылки
deactivate Backend

MobileApp --> User: Отображение экрана с ссылкой для дальнейшей отправки выбранный пользователем способ связи
deactivate MobileApp

@enduml
