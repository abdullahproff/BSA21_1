participant backend
database DB

activate User
User->MA: Клиент выбрал ингредиенты для фильтрации и нажал показать товары
activate MA
MA->backend: http-запрос на получение данных
activate backend

activate DB

deactivate DB
backend-->MA: http-ответ на запрос о получении данных
deactivate backend
MA-->User: Отображение экрана с результатом фильтрации
deactivate MA
