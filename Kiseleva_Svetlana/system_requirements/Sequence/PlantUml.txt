@startuml

autonumber

actor User as User
participant "MA" as Frontend
participant "Backend" as Backend
database "DB" as DataBase

activate User
User -> Frontend: Нажать на иконку выбора стикеров

activate Frontend
Frontend -> Backend: HTTP запрос списка стикеров

activate Backend
Backend -> Backend: Валидация запроса
Backend -> DataBase: SQL запрос списка стикеров

activate DataBase
DataBase --> Backend: Cписок стикеров
deactivate DataBase

Backend --> Frontend: Ответ с данными стикеров JSON
Frontend -> Frontend: Отобразить всплывающее окно выбора стикеров
User -> Frontend: Клик по стикеру
Frontend -> Backend: HTTP запрос на отправку стикера
Backend -> Backend: Сохранение информации о стикере
Backend --> Frontend: Ответ о подтверждение отправки JSON
deactivate Backend

Frontend -> User: Отобразить отправленный стикер
deactivate Frontend

@enduml
