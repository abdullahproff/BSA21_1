@startuml

autonumber

actor User as User
participant "MA" as Frontend
participant "Backend" as Backend
database "DB" as DataBase

activate User
User -> Frontend: Нажать на иконку выбора стикеров

activate Frontend
Frontend -> Frontend: Отобразить всплывающее окно выбора стикеров
User -> Frontend: Клик по стикеру
Frontend -> Backend: HTTP-запрос на отправку стикера

activate Backend
Backend -> Backend: Валидация запроса
Backend -> DataBase: Запрос данных стикера

activate DataBase
DataBase --> Backend: Ответ с данными стикера
deactivate DataBase

Backend --> Frontend: Возвращает JSON с подтверждением
deactivate Backend
Frontend -> User: Отображает отправленный стикер
deactivate Frontend
deactivate User

@enduml
