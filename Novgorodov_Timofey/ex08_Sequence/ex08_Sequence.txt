@startuml

autonumber

actor Student
participant WebApp

box "Server" #LightGray
participant Backend
database "DB (PgSQL)" as DB
end box

activate Student
Student->WebApp: создать задачу

activate WebApp
WebApp->Backend: https-запрос на добавление задачи

activate Backend
Backend->DB: SQL-запрос на запись данных

activate DB
DB-->Backend: возвращает набор данных \n(id созданной задачи)
deactivate DB
Backend-->WebApp: возвращает json с ответом \n(Статус 200 ОК)
deactivate Backend
WebApp-->Student: информация \nчто задача создана успешно
deactivate WebApp
deactivate Student

@enduml
