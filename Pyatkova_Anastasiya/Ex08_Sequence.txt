@startuml

actor User
participant MobileApp
participant backend
participant DB

autonumber

activate User
User -> MobileApp: Нажимает "Добавить товар к заказу"

activate MobileApp
MobileApp -> backend: HTTP-запрос на отображение каталога товаров

activate backend
backend -> DB: SQL-запрос на получение каталога товаров

activate DB
DB --> backend: Результат SQL-запроса с данными товаров
backend --> MobileApp: Каталог товаров
MobileApp --> User: Отображение каталога

User -> MobileApp: Выбирает товары
MobileApp -> backend: HTTP-запрос на добавление товаров в корзину

backend -> DB: SQL-запрос на добавление товаров в корзину
DB --> backend: Подтверждение обновления корзины

backend --> MobileApp: Обновление состава корзины
MobileApp --> User: Обновление корзины

User -> MobileApp: Переход в корзину
MobileApp --> User: Отображение корзины

User -> MobileApp: Нажимает "Оплатить добавленные товары"
MobileApp -> backend: HTTP-запрос на оплату заказа

backend -> DB: SQL-запрос Обновление оплаты и времени доставки
DB --> backend: Подтверждение оплаты

backend --> MobileApp: Ответ с подтверждением заказа и временем доставки
MobileApp --> User: Уведомление об успешной оплате

deactivate DB
deactivate backend
deactivate MobileApp
deactivate User

@enduml
