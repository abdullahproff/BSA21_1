@startuml

actor User
participant MobileApp
participant backend
participant DB

autonumber

activate User
User -> MobileApp: Нажимает "Оформить заказ"

activate MobileApp
MobileApp -> backend: HTTP-запрос на оформление заказа

activate backend
backend -> DB: SQL-запрос на обновление заказа и времени доставки

activate DB
DB --> backend: SQL-ответ с подтверждением оформления заказа
deactivate DB

backend --> MobileApp: HTTP-ответ с подтверждением заказа и временем доставки
deactivate backend

MobileApp --> User: Уведомление об успешном оформлении заказа

deactivate MobileApp
